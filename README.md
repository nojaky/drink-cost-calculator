# 음료 단가 계산기 Web

Roaster Hyun의 기존 Python/Tkinter 음료단가계산기를 웹용 정적 앱으로 업그레이드한 버전입니다.

## 주요 기능

- 내 재료 보관함: 재료명, 카테고리, 단위, 총량, 구매가격 등록
- 자동 단가 계산: 구매가격 / 총량 기준으로 g·ml·개당 단가 계산
- 레시피 원가 계산: 재료 선택 + 사용량 입력으로 원가 자동 계산
- 판매가 분석: 최소 판매가 배수, 적정 판매가 배수, 실제 판매가 기준 마진율 계산
- 레시피 저장/불러오기/복사/삭제
- 샘플 재료와 샘플 레시피 제공
- JSON 백업/불러오기
- 샘플 데이터와 JSON 불러오기 시 같은 재료/레시피는 덮어쓰기, 없는 항목만 추가
- 재료 CSV, 레시피 CSV 내보내기
- 서버 없이 브라우저 LocalStorage에 저장
- 개인정보 수집 없음


## 데이터 저장 방식

사용자의 재료와 레시피는 서버로 전송되지 않고, 현재 브라우저의 LocalStorage에 저장됩니다. 브라우저 데이터를 삭제하면 저장 내용도 사라질 수 있으므로 `전체 백업` 버튼으로 주기적으로 백업하는 것을 권장합니다.

## 기존 Python 버전 대비 변경점

- 엑셀 불러오기 중심 → 웹 내 재료 보관함 중심
- 엑셀 저장 중심 → JSON 백업 + CSV 내보내기 중심
- 레시피 단일 계산 → 레시피 저장/복사/불러오기 지원
- 판매가 배수 계산 + 실제 판매가 기준 마진율 계산 지원
